```
Пример:
Вывод: Введите число a
Ввод: 1
Вывод: Введите число b
Ввод: 2
Вывод: Результат сложения a и b: 3.0
Вывод: Результат вычитания a и b: -1.0
Вывод: Результат умножения a и b: 2.0
Вывод: Результат деления a и b: 0.5
```
